# 🏡 EFA Housing Society

A modern and responsive real estate website built using HTML, CSS, and JavaScript. This project showcases premium properties with an interactive user interface, property search functionality, image sliders, and detailed property modals.

## ✨ Features

- Responsive design for all screen sizes
- Modern and attractive user interface
- Interactive property listings
- Property image sliders
- Property search form
- Detailed property modal popup
- Smooth scrolling navigation
- Mobile-friendly navigation menu
- Testimonials section
- Contact form
- Scroll animations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
- Google Fonts

## 📂 Project Structure

```bash
EFA-Housing/
│── one.html
│── style.css
│── README.md
```

## 🚀 Getting Started

Follow these steps to run the project locally:

1. Clone the repository:

```bash
https://github.com/efafatima/Real-Estate-website.git
```

2. Navigate to the project folder:

```bash
cd EFA-Housing
```

3. Open `one.html` in your browser.

No additional installation or dependencies are required.
